# Android开发基础实验报告

## 项目简介
这是一个简单的Hello World Android应用程序，用于学习Android开发基础和环境搭建。

## 开发环境
- **开发工具**: Android Studio
- **开发语言**: Java
- **测试设备**: Pixel 2 XL

## 项目结构
MyApp/
├── app/
│ ├── src/main/
│ │ ├── java/ # 源代码
│ │ ├── res/ # 资源文件
│ │ └── AndroidManifest.xml
│ └── build.gradle # 模块配置
├── build.gradle # 项目配置
└── settings.gradle # 项目设置
